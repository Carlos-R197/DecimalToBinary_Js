# Requirements

- The program must be able to convert a number in base 10 to it's value in base 2.

# Acceptance Criteria

- The program must take a number in base 10 as a parameter and return it's base 2 version.

# Scenarios

1.1 '1' => '1'
1.2 '5' => '101'
1.3 '17' => '10001'
1.4 '106' => '1101010'
1.5 '1068' => '10000101100'
